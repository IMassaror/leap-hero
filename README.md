# Leap Hero

**Leap Hero** é um jogo de plataforma 2D com uma estética de arte 1-bit, desenvolvido pela *Take a Byte Studios*.

## Visão Geral

O jogo combina elementos de **Plataforma de Ação** (inspirado em *Shovel Knight*) e **Plataforma de Precisão** (inspirado em *Celeste*). O principal diferencial é a mecânica de transformação do personagem.

### Mecânicas Principais

O protagonista alterna entre duas formas:

  * **Guerreiro (Ação):** Focado na exploração de fases extensas e combate. Usa armadura e espada.
  * **Sapo (Precisão):** Focado na superação de obstáculos e movimentos precisos usando a língua.

## Stack Tecnológico

  * **Engine:** Unity (2022 LTS / 6)
  * **Linguagem:** C\#
  * **Arte:** Aseprite (Pixel Art 1-bit)
